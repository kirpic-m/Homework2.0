Проект на C++: Зв'язування файлів

Це базовий навчальний проект, що демонструє, як правильно зв'язувати два файли (main.cpp та std.cpp) в одній програмі на C++.
Основна ідея

Проект показує, як викликати функцію getText(), яка визначена в std.cpp, з головного файлу main.cpp. Це фундаментальний принцип модульного програмування на C++.
Структура файлів


Збірка та запуск
Необхідні інструменти

Компілятор C++, наприклад, g++.

Команди для компіляції


Скомпілювати main.cpp в об'єктний файл:

    
    g++ -c main.cpp -o main.o

  

Скомпілювати std.cpp в об'єктний файл:


    
    g++ -c std.cpp -o std.o

  

З'єднати об'єктні файли в одну програму:


        
    g++ main.o std.o -o my_program


Запуск програми

Після успішної компіляції запустіть виконуваний файл:

    
    ./my_program

  

Очікуваний результат у консолі:

    
    Цей текст з функції у std.cpp
